Despre proiect

Structura tehnică a aplicației

AutoTech Industrial este o aplicație web dezvoltată pentru prezentarea produselor de automatizare industrială și pentru simularea procesului de comandă online. Proiectul evidențiază arhitectura tehnică a unei platforme web, prin integrarea interfeței utilizator, a logicii backend și a bazei de date.

Arhitectură

Componentele principale ale sistemului

Interfața utilizator

Interfața aplicației este realizată cu HTML5, CSS3 și JavaScript, fiind organizată pe pagini dedicate pentru produse, autentificare, coș de cumpărături, checkout și contact.

Backend

Logica aplicației este implementată în Python folosind framework-ul Flask. Backend-ul gestionează rutele, autentificarea utilizatorilor, sesiunile, comenzile și legătura dintre interfață și datele stocate.

Baza de date

Aplicația utilizează o bază de date relațională pentru stocarea informațiilor despre produse, categorii, utilizatori, comenzi și mesaje transmise prin formular.

Tehnologii utilizate

Tehnologii și instrumente de dezvoltare

Frontend

HTML5 este utilizat pentru structura paginilor, CSS3 pentru stilizare și responsive design, iar JavaScript pentru interacțiunile dinamice din interfață.

Backend și procesare

Flask gestionează logica server-side, procesarea cererilor HTTP, afișarea paginilor dinamice și funcționalitățile asociate conturilor și comenzilor.

Persistența datelor

SQLite este utilizată pentru persistența datelor aplicației, oferind o soluție simplă și eficientă pentru implementarea proiectului.

Funcționalități

Funcționalități implementate

Gestionarea produselor

Utilizatorii pot vizualiza catalogul de produse, accesa paginile de detalii și analiza informațiile relevante pentru fiecare articol disponibil.

Cont utilizator și comandă

Platforma permite înregistrarea și autentificarea utilizatorilor, adăugarea produselor în coș și simularea procesului de finalizare a comenzii.

Administrare

Zona de administrare permite gestionarea produselor, categoriilor, comenzilor și mesajelor, separând funcționalitățile administratorului de cele ale clientului.
