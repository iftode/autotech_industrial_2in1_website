# AutoTech Industrial 2in1 Website

Modern industrial products e-commerce platform developed with Flask, SQLite, HTML, CSS and JavaScript.

## Project Overview

AutoTech Industrial 2in1 is a web platform designed for managing and selling industrial and automation products online.  
The application includes:

- Modern responsive interface
- Product catalog
- Categories management
- Shopping cart
- Checkout system
- User authentication
- Admin dashboard
- Orders management
- Invoice pages
- Contact page
- Product administration
- Dynamic templates using Jinja2

This project was developed as part of a personal portfolio and academic practice in web development and software engineering.
